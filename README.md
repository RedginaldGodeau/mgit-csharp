
# MGIT CSharp

Simplifie et réunis les commandes git pour gagné en productivité


## Documentation

### mgit [CMD] [ARGS]

#### mgit push [FILES]

    -p (ADD PATH / FILES / DIRECTORY)
    -m (COMMIT MESSAGE)
    -o (ORIGIN)

#### mgit clone [URL]
    -u (URL)
    -mk (ADDING SAVED DEVKEY)
    -k (ADDING DEVKEY)
